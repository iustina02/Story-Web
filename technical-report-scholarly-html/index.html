Introducere

Aplicatia web are rolul de a crea un mediu placut pentru citirea povestilor pentru copii in mediul online. utilizatorul are oportunitatea de a citi povesti cunoscut cat si povesti scrise de alti utilizatori sau sa-si creeze singur povestile, avand posibilitatea de a adauga imagini sugestive si chiar si o lista de personaje potrivita povestii. Toate progresele citirii povestilor se vor salva si se pot gasi usor pe profilul fiecarui utilizator.

Detalii

  • aplicatia web are mai multe pagini de desfasurare
  • are implemantata functionalitatea de a cauta anumite povesti dupa criteriile urmatoare
    • varsta
    • cuvinte cheie
  • used APIs:
    • The cat Api
  • aplicatia de foloseste de o baza de date SQL pentru a salva:
    • datele utilizatorilor
    • date legate de prezentarea povestilor
    • starea povestilor pentru fiecare utilizator
    • continutul paginiilor
  • tehnologii folosite: PHP 5, JavaScript, HTML, CSS, XAMPP
